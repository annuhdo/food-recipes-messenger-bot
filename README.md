## Configuration
- This bot requires a _config.js_ file in the main directory. A demo config file has been included. Replace appropriate strings.
- This bot runs on NodeJS and Heroku
- Once all the configurations are in place, use `heroku create` to create the webapp, and make sure to follow the [FB guide](https://developers.facebook.com/docs/messenger-platform) to synchronize a Facebook page to the app ğŸŸ.
